1. Buka github organisasi batch 
2. Buka repository tugas dan copy url
3. Buka terminal
4. pada terminal
```bash
git clone <copy>
cd <directory>
// KERJAKAN TUGAS PADA EDITOR
git checkout -b <nama branch>
git add <directory / file>
git commit -m "<message>"
git push origin <nama branch>
```
5. buat pull request
6. Assign buddy
